# Singular

|       | **m** | **f** | **n** |
| ----- | ----- | ----- | ----- |
| *nom* | -     | -a    | -o/-e |
| gen   | -a    | -e    | -a    |
| dat   | -u    | -i    | -u    |
| acc   | -/-a  | -u    | -o/-e |
| voc   | -e/-u | -o    | -o/-e |
| instr | -om   | -om   | -om   |
| loc   | -u    | -i    | -u    |
- masculine plural: short (one syllable) masculine names have their plural in -ovi, e.g.
	- voz -> vozovi
	- sir -> sirevi
	- sok -> sokovi
- masculine singular: 
	- inanimated stuff : acc = nom
	- animated stuff : acc = gen
